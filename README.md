## About
This website is used to monitor data from temperature, distance and humidity sensors

## Requirements
-Arduiono
-DHT 22
-HCSR 04
-Breadboard
-Jumper cables ( Male-male and Female-female )
-USB to USB cable

## Installation
git clone https://github.com/Daarns/Website-IOT.git
cd Website-IOT
